# 机器学习任务定义

- 知识点：监督学习、无监督学习、分类、回归、聚类、目标变量、评价指标

开始建模前，最重要的不是先选算法，而是先定义任务。机器学习任务通常可以分为三类：回归、分类和无监督学习。回归预测连续值，例如房价、销售额、温度；分类预测离散类别，例如是否购买、是否患病、图片属于哪一类；无监督学习没有明确标签，常见目标是发现样本结构，例如聚类、降维和异常发现。

定义任务时要先明确目标变量是什么、特征是什么、样本粒度是什么。例如一张学生成绩表里，行通常是学生，列是年龄、出勤率、作业得分等变量；如果目标是预测期末成绩，这是回归；如果目标是预测是否及格，这是分类。任务不同，后续选择的模型、损失函数和评价指标都会不同。

教学上应引导学习者先回答三个问题：要预测什么？输入信息有哪些？成功标准是什么？如果没有明确评价指标，模型训练就会变成“跑代码看分数”，无法判断结果是否真的有意义。

实操要点：

```python
# 分类任务：y 是类别
X = df.drop(columns=['is_pass'])
y = df['is_pass']

# 回归任务：y 是连续值
X = df.drop(columns=['score'])
y = df['score']
```

易错点：
- 不要把 ID、姓名、时间戳等无意义字段直接当成强特征。
- 不要在任务没定义清楚前直接套模型。
- 分类和回归的指标不能混用，分类不用 RMSE，回归也不用 accuracy。
