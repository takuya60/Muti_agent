# 前向传播与反向传播

- 知识点：forward、backpropagation、链式法则、梯度、参数更新

前向传播是神经网络从输入到输出的计算过程：每一层用当前参数计算结果，最后得到预测值和损失。反向传播则从损失出发，利用链式法则计算每个参数对损失的影响，也就是梯度。优化器根据梯度更新参数，让下一次预测更接近真实标签。

可以把训练过程理解为一个循环：前向计算预测，计算损失，反向计算梯度，更新权重。这个思想与传统机器学习中的梯度下降一脉相承，只是神经网络层数更多、参数更多。

教学中不必让初学者手推复杂矩阵导数，但应理解为什么需要可导函数、为什么激活函数会影响梯度传播。

```python
# PyTorch 风格训练循环示意
optimizer.zero_grad()
y_pred = model(X_batch)
loss = criterion(y_pred, y_batch)
loss.backward()
optimizer.step()
```

易错点：
- 反向传播不是“反着再预测一次”，而是计算梯度。
- 忘记清空梯度会导致梯度累积。
- 学习率过大可能震荡，过小训练很慢。
