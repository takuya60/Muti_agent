# 异常值识别与处理

- 知识点：箱线图、分位数、业务规则、稳健处理、异常值不等于错误值

异常值是明显偏离大多数样本的观测值，但异常值不一定是错误值。比如收入数据中的极高收入、交易金额中的大额订单，都可能是真实且重要的样本。处理异常值前必须结合业务含义判断：是录入错误、极端但真实、还是对模型有影响的长尾样本。

常见识别方法包括箱线图、分位数范围、Z-score 和业务规则。对于初学者，可以先用分位数查看极端区间，再配合可视化理解分布。

```python
import seaborn as sns

print(df['income'].describe(percentiles=[0.01, 0.05, 0.95, 0.99]))
sns.boxplot(x=df['income'])

lower = df['income'].quantile(0.01)
upper = df['income'].quantile(0.99)
df['income_clip'] = df['income'].clip(lower, upper)
```

异常值处理方式包括删除、截断、分箱、对数变换、使用对异常值不敏感的模型或指标。不能为了让模型分数看起来好，就机械删除所有极端样本。

审核规则：
- 必须说明异常值判断依据。
- 不建议无解释地删除大量样本。
- 对测试集不能用测试集自身分位数重新确定截断边界，边界应来自训练集。
