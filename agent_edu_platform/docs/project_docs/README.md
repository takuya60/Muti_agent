# AgentEdu 团队协作文档中心

本目录包含项目团队并行开发所需的全套文档。每位成员在开始工作前，请先阅读与自己职责对应的文档。

## 文档索引

| 编号 | 目录 | 内容 | 目标读者 |
|------|------|------|---------|
| 01 | `01-requirements/` | 产品需求规格书 (PRD)、功能清单、用户故事 | 全员 |
| 02 | `02-architecture/` | 系统架构图、模块职责、数据流、部署拓扑 | 全员 |
| 03 | `03-tech-stack/` | 前端/后端/Agent/数据库/LLM 技术选型依据 | 技术负责人 |
| 04 | `04-api-contracts/` | 前后端接口、Agent 内部接口、LLM 调用规范 | 前端/后端/Agent 开发者 |
| 05 | `05-database/` | ER 图、表结构定义、字段说明、索引策略 | 后端开发者 |
| 06 | `06-development-guide/` | 环境搭建、Git 分支策略、代码规范、提交规范 | 全员 (尤其新人) |
| 07 | `07-testing/` | 测试策略、用例设计、验收标准、性能基准 | 测试人员 |
| 08 | `08-archive/` | 早期规划文档 | 项目归档/答辩材料 |

## 文档维护规范

- **修改前**：先在群组通知，避免并行修改冲突
- **修改后**：在 Git Commit Message 中标注 `docs: <改动描述>`
- **版本控制**：所有文档均纳入 Git 管理，禁止在本地私存不同版本
