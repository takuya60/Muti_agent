\textbf{题目：}

利用分离变量法求解一维热传导方程的初边值问题：

$$ \frac{\partial u}{\partial t} = \alpha^2 \frac{\partial^2 u}{\partial x^2}, \quad 0 < x < L, \quad t > 0 $$

边值条件为：
$$ u(0, t) = 0, \quad u(L, t) = 0, \quad t > 0 $$

初值条件为：
$$ u(x, 0) = f(x) = \begin{cases} x, & 0 \leq x \leq L/2 \\ L - x, & L/2 < x \leq L \end{cases} $$

要求：
1. 写出分离变量的假设和特征值问题的推导过程。
2. 求解出傅里叶级数展开式的系数。
3. 给出 $u(x, t)$ 的最终级数解表达式。
